# Python for Data Analysis

This repository contains Python programs developed to strengthen core programming concepts required for data analysis and business analytics roles.

## Concepts Covered
- Object-Oriented Programming (Inheritance, Polymorphism, Encapsulation)
- Control structures (if-else, for loops, while loops)
- Data handling using matrices
- Type casting and logical operations

## Key Outcomes
- Built strong logical and analytical problem-solving skills  
- Applied Python programming concepts relevant to data analysis workflows  
- Strengthened coding foundations for analytics, automation, and reporting tasks
